# 开发环境配置

## 1.scala

需要在本地安装scala和sbt

以archlinux为例

```shell
yay -S scala
yay -S sbt
yay -S verilator
```


## 2. IDE选择

- vscode 界面美观，但是报错能力弱
- Intell 笨重但是报错补全比较行
